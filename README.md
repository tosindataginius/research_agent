## 🤖 Autonomous Research Assistant Agent

An autonomous AI application that takes a research topic, generates a multi-step query plan, scrapes high-quality live data from the web, and compiles a comprehensive, structured markdown report.

Built using **Python, LangChain, Streamlit,** and powered by the **Google Gemini API.**


## Key Features

**Intent Planning:** Breaks down complex user requests into granular research sub-topics.

**Live Web Search:** Uses the updated ddgs library to scrape fresh content from the web.

**Rate-Limit Resilience:** Embedded exponential backoff structures to handle Free-Tier Google API limits (429 Resource Exhausted) safely.

**Streamlit UI:** Clean, interactive dashboard with real-time process monitoring.

**Clean Exports:** Generates production-ready, beautifully structured Markdown research summaries.
